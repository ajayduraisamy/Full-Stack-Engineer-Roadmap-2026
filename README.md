## 🎯 Mission

**Commit 90 days. Build real projects. Ship production-ready skills.**

This roadmap is divided into **3 monthly sprints**:

- **Month 1 (Days 1-30):** Modern Frontend
- **Month 2 (Days 31-60):** Logic & Frameworks
- **Month 3 (Days 61-90):** Backend & Scalability

---

## 🧠 Sprint Breakdown

### 🟦 Month 1: The Modern Frontend (Days 1-30)

**Focus Areas:**
- **Semantic HTML5**
- **CSS Mastery** (Flexbox, Grid, Responsive Design)
- **JavaScript Deep Dive** (DOM, ES6+, Async/Await)
- **Tailwind CSS**

**Milestone Project:**
- **Responsive Portfolio Site** (Vanilla JS + Tailwind CSS)

### 🟨 Month 2: The Logic & Frameworks (Days 31-60)

**Focus Areas:**
- **React.js** (Components, Hooks, Context API)
- **Next.js** (App Router, SSR, SSG)
- **TypeScript** (Types, Interfaces, Utility Types)
- **State Management**

**Milestone Project:**
- **High-Performance Dashboard** consuming a public API (Crypto or Movies)

### 🟩 Month 3: The Backend & Scalability (Days 61-90)

**Focus Areas:**
- **Node.js + Express**
- **PostgreSQL + Prisma ORM**
- **JWT Authentication**
- **Docker Basics**

**Final Capstone:**
- **Full-Stack SaaS Starter** with Auth, Database, and Deployment
